1. ЭЛЕМЕНТ <HEAD> И МЕТАДАННЫЕ ВЕБ-СТРАНИЦЫ

Базовые задачи:

  1. Создать страницу с метаданными

    • Входные данные: Название страницы: "Моя первая страница"
    • Решение:
      html
      <!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Моя первая страница</title>
</head>
<body>
</body>
</html>

  2. Добавить описание страницы

    • Входные данные: Описание: "Это пример страницы."
    • Решение:
      html
      <meta name="description" content="Это пример страницы.">

  3. Добавить ключевые слова

    • Входные данные: Ключевые слова: "HTML, пример, задача"
    • Решение:
      html
      <meta name="keywords" content="HTML, пример, задача">

  4. Добавить ссылку на CSS файл

    • Входные данные: Файл: "styles.css"
    • Решение:
      html
      <link rel="stylesheet" href="styles.css">

  5. Добавить JavaScript файл

    • Входные данные: Файл: "script.js"
    • Решение:
      html
      <script src="script.js"></script>

  6. Задать кодировку страницы

    • Входные данные: Кодировка: "UTF-8"
    • Решение:
      html
      <meta charset="UTF-8">

  7. Добавить авторские права

    • Входные данные: Автор: "Ваше Имя"
    • Решение:
      html
      <meta name="author" content="Ваше Имя">

  8. Добавить страницу к списку избранных

    • Решение:
      html
      <link rel="icon" href="favicon.ico" type="image/x-icon">

  9. Задать параметры кэширования

    • Решение:
      html
      <meta http-equiv="Cache-Control" content="no-cache, no-store, must-revalidate">

  10. Указать характер страницы

    • Решение:
      html
      <meta name="robots" content="index, follow">

Сложные задачи:

  1. Создать страницу с полной мета-информацией

    • Входные данные: Название, описание, ключевые слова.
    • Решение:
      html
      <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta name="description" content="Это пример страницы.">
    <meta name="keywords" content="HTML, пример, задача">
    <meta name="author" content="Ваше Имя">
    <title>Моя первая страница</title>
</head>

  2. Использовать Open Graph метаданные

    • Входные данные: URL: "https://example.com", название: "Пример", описание: "Пример страницы."
    • Решение:
      html
      <meta property="og:title" content="Пример">
<meta property="og:description" content="Пример страницы.">
<meta property="og:url" content="https://example.com">

  3. Добавить мета-тег для Twitter Cards

    • Входные данные: Заголовок: "Заголовок страницы", описание: "Описание страницы."
    • Решение:
      html
      <meta name="twitter:title" content="Заголовок страницы">
<meta name="twitter:description" content="Описание страницы.">

  4. Добавить кросс-доменные метаданные

    • Решение:
      html
      <meta http-equiv="Access-Control-Allow-Origin" content="*">

  5. Установить viewport для мобильных устройств

    • Решение:
      html
      <meta name="viewport" content="width=device-width, initial-scale=1">

  6. Добавить авторские права с датой

    • Входные данные: Год: "2024", Автор: "Ваше Имя"
    • Решение:
      html
      <meta name="copyright" content="Copyright 2024, Ваше Имя">

  7. Добавить мета-тег для языковых настроек

    • Решение:
      html
      <meta http-equiv="Content-Language" content="ru">

  8. Использовать мета-тег для защиты от кражи контента

    • Решение:
      html
      <meta name="googlebot" content="noindex, nofollow">

  9. Добавить ссылку на альтернативную версию страницы

    • Входные данные: Альтернативная версия: "https://example.com/alternative"
    • Решение:
      html
      <link rel="alternate" href="https://example.com/alternative" hreflang="ru">

  10. Создать мета-тег для viewport с масштабированием

    • Решение:
      html
      <meta name="viewport" content="width=device-width, initial-scale=1.0, maximum-scale=1.0, user-scalable=no">

2. ЭЛЕМЕНТЫ ГРУППИРОВКИ

Базовые задачи:

  1. Создать абзац текста

    • Входные данные: Текст: "Это мой первый абзац."
    • Решение:
      html
      <p>Это мой первый абзац.</p>

  2. Создать раздел с заголовком и параграфом

    • Входные данные: Заголовок: "Мой заголовок", Текст: "Это текст под заголовком."
    • Решение:
      html
      <h1>Мой заголовок</h1>
<p>Это текст под заголовком.</p>

  3. Создать блок с использованием <div>

    • Входные данные: Текст: "Текст внутри блока."
    • Решение:
      html
      <div>Текст внутри блока.</div>

  4. Создать блок с отступами

    • Решение:
      html
      <div style="margin: 20px;">Текст с отступами.</div>

  5. Использовать <span> для выделения текста

    • Входные данные: Текст: "Выделенный текст"
    • Решение:
      html
      <p>Это <span style="font-weight:bold;">выделенный текст</span> в абзаце.</p>

  6. Создать список с элементами

    • Входные данные: Элементы: "Первый", "Второй", "Третий"
    • Решение:
      html
      <ul>
    <li>Первый</li>
    <li>Второй</li>
    <li>Третий</li>
</ul>

  7. Создать контейнер с несколькими параграфами

    • Входные данные: Параграфы: "Первый", "Второй"
    • Решение:
      html
      <div>
    <p>Первый параграф.</p>
    <p>Второй параграф.</p>
</div>

  8. Создать таблицу с заголовками

    • Входные данные: Заголовки: "Имя", "Возраст"
    • Решение:
      html
      <table>
    <tr>
        <th>Имя</th>
        <th>Возраст</th>
    </tr>
</table>

  9. Создать раздел с цитатой

    • Входные данные: Цитата: "Это цитата."
    • Решение:
      html
      <blockquote>Это цитата.</blockquote>

  10. Создать элемент <strong> для выделения текста

    • Входные данные: Текст: "Важно!"
    • Решение:
      html
      <p>Это <strong>Важно!</strong> сообщение.</p>

Сложные задачи:

  1. Создать вложенные элементы <div>

    • Входные данные: Внешний текст: "Внешний", Внутренний текст: "Внутренний"
    • Решение:
      html
      <div>
    Внешний
    <div>Внутренний</div>
</div>

  2. Использовать несколько <span> для форматирования

    • Входные данные: "Этот текст <bold>жирный</bold> и <italic>курсивный</italic>."
    • Решение:
      html
      <p>Этот текст <span style="font-weight:bold;">жирный</span> и <span style="font-style:italic;">курсивный</span>.</p>

  3. Создать блочный элемент с фоном и отступами

    • Входные данные: Текст: "Содержимое блока"
    • Решение:
      html
      <div style="background-color: lightgray; padding: 20px;">Содержимое блока</div>

  4. Создать многоуровневый список

    • Входные данные: Список: "Первый", "Второй", "Третий" (с подсписком "A", "B")
    • Решение:
      html
      <ul>
    <li>Первый</li>
    <li>Второй
        <ul>
            <li>A</li>
            <li>B</li>
        </ul>
    </li>
    <li>Третий</li>
</ul>

  5. Создать элемент <header> с заголовком и навигацией

    • Решение:
      html
      <header>
    <h1>Заголовок</h1>
    <nav>
        <ul>
            <li><a href="#home">Главная</a></li>
            <li><a href="#about">О нас</a></li>
        </ul>
    </nav>
</header>

  6. Создать элемент <footer> с контактной информацией

    • Решение:
      html
      <footer>
    <p>Контактная информация: email@example.com</p>

  7. Использовать <main> для основного контента

    • Решение:

    <h2>Основной контент</h2>

  8. Создать адаптивный контейнер с изображением и текстом

    • Решение:
      html

    <img src="image.jpg" alt="Описание изображения" style="width: 100px;">

  9. Создать <article> с заголовком и содержимым

5. РАБОТА С ИЗОБРАЖЕНИЯМИ

Базовые задачи:

  1. Добавить изображение с помощью тега <img>

    • Входные данные: URL: "image.jpg", Альтернативный текст: "Описание изображения"
    • Решение:
      html
      <img src="image.jpg" alt="Описание изображения">

  2. Добавить изображение с заданной шириной и высотой

    • Входные данные: URL: "image.jpg", Ширина: 300px, Высота: 200px
    • Решение:
      html
      <img src="image.jpg" alt="Описание изображения" width="300" height="200">

  3. Добавить изображение с фоном

    • Решение:
      html
      <div style="background-image: url('image.jpg'); width: 300px; height: 200px;"></div>

  4. Добавить изображение с ссылкой

    • Входные данные: URL: "image.jpg", Ссылка: "https://example.com"
    • Решение:
      html
      <a href="https://example.com"><img src="image.jpg" alt="Описание изображения"></a>

  5. Добавить изображение с классом для стилизации

    • Входные данные: URL: "image.jpg", Класс: "my-image"
    • Решение:
