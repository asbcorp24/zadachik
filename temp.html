1. Загрузка видео и субтитров с YouTube

Базовые задачи

  1. Загрузка видео по ссылке

    • Ситуация: Вам нужно скачать видео с YouTube для просмотра оффлайн.
    • Пример:
      python
      from pytube import YouTube

url = 'https://www.youtube.com/watch?v=example'
yt = YouTube(url)
yt.streams.first().download()
    • Решение: Программа загружает первое доступное качество видео.

  2. Выбор качества видео

    • Ситуация: Вы хотите загрузить видео в высоком качестве.
    • Пример:
      python
      yt.streams.filter(progressive=True, file_extension='mp4').order_by('resolution').desc().first().download()
    • Решение: Загружает видео в самом высоком качестве.

  3. Загрузка субтитров

    • Ситуация: Вы хотите скачать субтитры для видео.
    • Пример:
      python
      yt.captions['en'].download(title=yt.title)
    • Решение: Загружает английские субтитры и сохраняет их под названием видео.

  4. Загрузка видео с указанием пути

    • Ситуация: Вам нужно сохранить видео в конкретной папке.
    • Пример:
      python
      yt.streams.first().download(output_path='/path/to/folder')
    • Решение: Загружает видео в указанную директорию.

  5. Вывод информации о видео

    • Ситуация: Перед загрузкой нужно узнать детали о видео.
    • Пример:
      python
      print(f'Название: {yt.title}, Длительность: {yt.length} секунд')
    • Решение: Выводит название и длительность видео.

  6. Загрузка видео и субтитров одновременно

    • Ситуация: Вы хотите загрузить видео и его субтитры одновременно.
    • Пример:
      python
      yt.streams.first().download()
yt.captions['en'].download(title=yt.title)
    • Решение: Загружает видео и субтитры последовательно.

  7. Загрузка видео с проверкой существования файла

    • Ситуация: Вы хотите избежать перезаписи существующего файла.
    • Пример:
      python
      import os

    yt.streams.first().download()
else:
    print('Файл уже существует.')
    • Решение: Проверяет, существует ли файл, перед загрузкой.

  8. Загрузка видео по ID

    • Ситуация: У вас есть только ID видео, а не полная ссылка.
    • Пример:
      python
      video_id = 'example'

yt.streams.first().download()
    • Решение: Сначала формируется полная ссылка, затем загружается видео.

  4. Загрузка видео с логированием

    • Ситуация: Вы хотите логировать процесс загрузки в файл.

  5. Создание списка для загрузки плейлистов
